# 05 Caching And Redis

## Overview
This phase introduces the principles of caching and the deep architecture of Redis, heavily focused on the MERN (MongoDB, Express, React, Node.js) stack. You will learn how to protect your MongoDB instances and accelerate your Node.js APIs using advanced in-memory data structures.

## Topics Covered
- [01 Caching Fundamentals (MERN Stack Edition)](./01-caching-fundamentals.md)
- [02 Redis Architecture and Advanced Data Structures (MERN Stack Edition)](./02-redis-architecture.md)

## Learning Objectives
- Understand the physics and architecture of computer memory in the context of Node.js (V8) and MongoDB.
- Learn why databases become bottlenecks and why caching is the ultimate solution.
- Master the cache hit vs cache miss concept and Cache Aside pattern using Express.js.
- Deep dive into Redis, its single-threaded event loop, and I/O multiplexing.
- Master advanced Redis data structures (Streams, Bitmaps, HyperLogLog, Pub/Sub) and persistence (RDB/AOF).
- Learn how to implement Redis safely in a Node.js production environment (using `ioredis`).

## Prerequisites
- Basic understanding of web applications, RAM, and databases.
- Familiarity with Node.js, Express.js, and MongoDB/Mongoose.
- Completion of Phase 04 (Database Engineering).
