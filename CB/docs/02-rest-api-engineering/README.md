# Phase 02: REST API Engineering

Welcome to Phase 2. Now that you understand the Internet and HTTP, it's time to build the actual "contracts" that power the web. This phase dives deep into REST API design, covering everything from basic routing to professional pagination, filtering, sorting, and versioning.

---

## 🎯 Learning Objectives

By the end of this phase, you will be able to:
- Design intuitive, noun-based RESTful resources.
- Implement scalable pagination (offset and cursor-based).
- Build complex filtering and sorting mechanics securely.
- Version your APIs to prevent breaking changes for existing clients.
- Structure error handling professionally.
